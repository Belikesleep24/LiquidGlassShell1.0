INSTRUCCIONES PARA AGREGAR SF PRO ROUNDED
==========================================

IMPORTANTE: El nombre del archivo NO importa, pero el nombre de la FAMILIA dentro del archivo SÍ.

1. Descarga la fuente SF Pro Rounded (archivos .ttf o .otf)

2. Coloca los archivos de la fuente en esta carpeta (Fonts/)

3. El nombre del archivo puede ser cualquier cosa, por ejemplo:
   - SF-Pro-Rounded-Regular.ttf
   - SFProRounded-Regular.ttf
   - fuente.ttf
   - etc.

4. Lo que SÍ importa es que el nombre de la FAMILIA dentro del archivo de fuente sea:
   "SF Pro Rounded"
   
   Para verificar el nombre de la familia:
   - Abre el archivo .ttf o .otf
   - El nombre que aparece es el nombre de la familia
   - Debe decir exactamente "SF Pro Rounded" (sin variaciones)

5. El código buscará automáticamente cualquier archivo en esta carpeta que tenga
   el nombre de familia "SF Pro Rounded"

6. Si la fuente no se encuentra, la aplicación usará la fuente del sistema por defecto

NOTA: Si tienes múltiples variantes (Regular, Bold, Light, etc.), puedes ponerlas todas
en esta carpeta. WPF las detectará automáticamente.
